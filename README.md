# Universal Calculator

A modular, open-source calculator engine built with Python.
Designed to cover arithmetic, advanced math, physics formulas, and developer utilities in one extensible package.

## Current Status

**v1.0 - Arithmetic Core: COMPLETE** ✅
Implemented a complete arithmetic module using SymPy for exact calculations with Rational numbers.

### Features in v1.0
- **Basic Ops**: `+ - * /` with exact fraction support
- **Advanced**: `^ power`, `sqrt`, `log`, `ln`, `e^x`
- **Utility**: `abs`, `ceil`, `floor`, `round`, `min`, `max`, `mod`, `factorial`
- **Error Handling**: Division by zero protection

## Project Structure

- [x] `core/arithmetic.py` - Basic math engine
- [ ] `core/number_theory.py` - GCD, LCM, Prime
- [ ] `core/trigonometry.py` - sin, cos, tan
- [ ] `core/calculus.py` - derivative, integral
- [ ] `core/statistics.py` - mean, median, std
- [ ] `core/physics.py` - formulas
- [ ] `main.py` - CLI interface

## Tech Stack
- **Language**: Python 3
- **Library**: SymPy for symbolic math
- **Goal**: Replace Casio FX with code
